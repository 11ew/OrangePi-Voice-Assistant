## 📦 版本规划

### V1.0 基础版（最简单，快速验证）⚡

**目标：** 30分钟内跑通完整对话流程

**功能：**
- ✅ 按键触发录音（5秒固定时长）
- ✅ 云端语音识别（Whisper API 或 百度/阿里云）
- ✅ DeepSeek对话（已有）
- ✅ Edge-TTS语音合成（已有）
- ✅ 播放语音

**架构：**
```
按键触发 → 录音5秒 → 云端ASR → DeepSeek → TTS → 播放
```

**优点：**
- 🚀 最快实现
- 🔧 无需本地模型
- 📝 代码最简单

**缺点：**
- ⏱️ 延迟较高（10-15秒）
- 💰 需要API密钥（可能有费用）

**实施内容：**
1. 创建 `scripts/audio_record.py` - 录音模块
2. 创建 `scripts/speech_to_text_api.py` - 云端ASR
3. 创建 `voice_chat_v1.sh` - 主程序
4. 测试完整流程

**验收标准：**
- ✅ 能够录音并播放
- ✅ 能够识别语音并对话
- ✅ 完整流程运行正常

---

### V2.0 本地ASR版（降低延迟）⚡⚡

**目标：** 使用本地Whisper，降低延迟和成本

**新增功能：**
- ✅ 本地Whisper识别（Faster-Whisper）
- ✅ 模型自动下载
- ✅ 配置文件管理

**架构：**
```
按键触发 → 录音5秒 → 本地Whisper → DeepSeek → TTS → 播放
```

**优点：**
- ⚡ 延迟降低（8-10秒）
- 💰 无ASR费用
- 🔒 隐私保护

**改进：**
- 识别延迟从 3秒 降到 1-2秒
- 不依赖外部ASR服务

**实施内容：**
1. 安装 `faster-whisper`
2. 创建 `scripts/speech_to_text_local.py` - 本地ASR
3. 创建 `scripts/download_models.sh` - 模型下载
4. 更新 `voice_chat_v2.sh` - 支持本地ASR

**验收标准：**
- ✅ 本地Whisper识别准确率 > 90%
- ✅ 识别延迟 < 2秒
- ✅ 完全离线识别

---

### V3.0 VAD连续对话版（更自然）⚡⚡⚡

**目标：** 实现连续对话，无需按键

**新增功能：**
- ✅ VAD语音活动检测（Silero-VAD）
- ✅ 自动检测语音开始/结束
- ✅ 连续对话模式

**架构：**
```
持续监听 → VAD检测 → 自动录音 → 本地Whisper → DeepSeek → TTS → 播放 → 返回监听
```

**优点：**
- 🎭 更自然的对话体验
- 🔄 无需手动触发
- ⏱️ 响应更快

**改进：**
- 支持自然对话流程
- 自动检测说话开始和结束

**实施内容：**
1. 安装 Silero-VAD
2. 创建 `scripts/vad_detector.py` - VAD检测
3. 创建 `scripts/voice_controller.py` - 状态机控制
4. 更新 `voice_chat_v3.sh` - 支持连续对话

**验收标准：**
- ✅ VAD准确检测语音开始/结束
- ✅ 连续对话流畅
- ✅ 误触发率 < 5%

---

### V4.0 流式输出版（终极目标）🚀🚀🚀

**目标：** 实现流式对话，边生成边播放

**新增功能：**
- ✅ DeepSeek流式对话
- ✅ 句子分割器
- ✅ 并行TTS生成
- ✅ 流式音频播放

**架构：**
```
持续监听 → VAD → Whisper → DeepSeek流式 → 句子分割 → 并行TTS → 流式播放
                                    ↓
                              边生成边播放
```

**优点：**
- ⚡ 首字延迟 < 1秒
- 🎭 类似真人对话
- 🚀 极致体验

**改进：**
- 总延迟从 8秒 降到 3秒
- 首字延迟 < 1秒
- 边说边播

**实施内容：**
1. 修改 `scripts/deepseek_chat.py` - 支持流式输出
2. 创建 `scripts/sentence_splitter.py` - 句子分割
3. 创建 `scripts/streaming_tts.py` - 流式TTS
4. 创建 `scripts/streaming_player.py` - 流式播放
5. 创建 `voice_chat_v4.sh` - 流式对话主程序

**验收标准：**
- ✅ 首字延迟 < 1秒
- ✅ 总延迟 < 3秒
- ✅ 流畅无卡顿
