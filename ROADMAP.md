## 🎯 项目概述

**EdgeTTSdemo** 是一个运行在 Orange Pi AI Pro 上的**完全离线实时语音助手系统**。

### 核心特性

- 🎤 **实时语音识别**：基于 Paraformer 模型
- 🗣️ **智能对话**：集成 DeepSeek LLM
- 🔊 **语音合成**：使用 Edge-TTS
- 🎯 **语音活动检测**：Silero VAD（已优化）
- 💻 **边缘计算**：完全离线运行，保护隐私
- ⚡ **低延迟**：优化的实时处理流程

**关键文件：**
- `realtime_assistant/vad_detector.py`
- `config/vad_config.json`
- `scripts/test_vad.py`
- `scripts/test_vad_energy.py`
- `realtime_assistant/rnnoise_denoiser.py`

**关键文档：**
- [VAD 优化研究报告](docs/vad-optimization-report.md) ⭐

**重要发现：**
1. **音量反向特性**：Orange Pi 音频驱动音量值越小，录音信号越强
2. **sherpa-onnx 框架**：深入理解了 sherpa-onnx 与官方 Silero VAD 的差异
3. **VAD 配置优化**：threshold=0.2, min_silence_duration=0.5s
4. **RNNoise 结论**：不推荐在当前硬件环境使用（信号衰减 53%）

**完成日期：** 2026-01-19

## 📝 技术债务

### 已知问题

1. **VAD 片段检测**
   - 当前检测到 2 个片段，实际有 3 个
   - 已确认这是合理的（语义完整性）
   - 备选方案：能量 VAD（如需更精细分割）

2. **音频格式限制**
   - 必须使用 RAW 格式录音，然后转换为 WAV
   - Orange Pi 驱动的 WAV 格式录音有 bug

3. **sherpa-onnx 参数限制**
   - 缺少 neg_threshold 和 speech_pad_ms 参数
   - 如需更多控制，可考虑切换到官方 Silero VAD
